## 체스♟

### 게임

- [x] 게임은 양팀이  한번씩 진행
- [x] 상대방 킹을 잡을 경우 게임 승리

### 체스말

1. 킹
    - [x] 방향은 8방향으로 1칸 제한
    - [x] 죽는다면 게임 종료
2. 퀸
    - [x] 방향은 8방향 칸 제한 없음
3. 룩
    - [x] 상하좌우 칸 제한 없음
4. 비숍
    - [x] 4방향 대각선 칸 제한 없음
5. 나이트
    - [x] 4방향 중 1개 2칸 + 좌 or 우 1칸
6. 폰
    - [x] 기본 움직임 앞으로 1칸
    - [x] 시작선에서 2칸
    - [x] 상대방 말이 대각선에 있을 경우 대각선으로 이동
